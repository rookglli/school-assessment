# School Assessment

Веб-програма для перевірки засвоєння дітьми шкільної програми.

## Можливості проекту

- реєстрація та авторизація користувачів;
- cтворення профілю дитини;
- вибір класу та шкільного предмета;
- проходження тестів;
- підрахунок відсотка засвоєння матеріалу;
- визначення слабких тем;
- персональні рекомендації;
- збереження історії результатів.

## Структура

- `client` - frontend;
- `server` - backend;
- `docs` - документація проекту.
